# Tests E2E

## Configuração 

1. Instalar a versão recente do JDK;
2. Instalar o python 2.X;
3. Instalar Node.js; 
4. Execute

```sh
npm install
```

## 

1. Iniciar o selenium webdriver
```sh
npm start
```

2. Rodar os testes
```sh
npm test
```

3. Rodar uma suíte de testes específica
```sh
npm test --suite smoke
npm test --suite stress
npm test --suite all
```

2. Rodar apenas Smoke test
```sh
npm test --suite smoke
ou 

```
